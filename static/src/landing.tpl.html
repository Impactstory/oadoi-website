oaDOI is a comprehensive index of fulltext options for 90 million scholarly articles, accessible with a free, fast, open API.

Our API powers Unpaywall, a free browser extension that lets you bypass paywalls on scholarly articles.

For libraries, we support an SFX integration that finds open fulltext when there's no subscription access; more than 600 libraries use it worldwide.
